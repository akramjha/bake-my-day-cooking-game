# Bake My Day

## Overview

Bake My Day is a 2D cooking simulation game developed using Unity and C#. The game is inspired by Cooking Mama-style gameplay, where players complete cooking steps, choose the correct ingredients, and prepare recipes through interactive challenges.

The project focuses on simple cooking mechanics, time-based tasks, step-by-step gameplay, and user-friendly interaction suitable for casual players.

---

## Key Highlights

- Developed using Unity and C#
- Cooking simulation gameplay
- Step-by-step recipe completion
- Timer-based challenges
- Interactive ingredient selection
- Level-based game progression

---

## Features

- Main menu with Play and Settings
- Multiple cooking difficulty levels
- Ingredient selection system
- Step-by-step cooking process
- Timer for each cooking task
- Life system for failed attempts
- Restart system when lives are lost
- Interactive UI feedback

## Gameplay Flow

1. Player starts from the main menu
2. Player selects a level
3. Player chooses the correct ingredients
4. Player completes cooking steps within the timer
5. Player loses a life if they fail a step
6. Player completes the recipe to progress to the next level
